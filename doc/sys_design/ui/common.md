# 共通レイアウト・コンポーネント定義

全画面で共通して使うパーツの仕様。SP/PC で寸法のみ変更し、見た目は統一する。

---

## 1. ヘッダー (Header)

### 役割
- サイト名（ロゴ）の表示
- 主要アクション（例: New post）の配置
- 全画面で固定表示（sticky）を推奨

### 仕様

| 項目 | SP | PC |
|------|-----|-----|
| 高さ | 56px | 64px |
| 背景 | `--card`（shadcn） |
| ボーダー | 下線 0.5px `--border` |
| 左右パディング | 24px | 48px |

### 配置
- **左**: サイト名「jeligramy」  
  - フォント: system-ui, 18px(SP) / 20px(PC), font-weight 600, 色 `#fafafa`
- **右**: プライマリボタン（例: New post）  
  - 高さ 32px(SP) / 36px(PC), 角丸 `--radius-md`（8px）  
  - 背景 `--primary`, 文字 `--primary-foreground`
  - フォント 13px(SP) / 14px(PC)

### 使用画面
- トップ、新規投稿、その他全画面

---

## 2. サイドメニュー (Side Menu)

### 役割
- 主要ナビゲーション（ホーム、新規投稿、プロフィールなど）の配置
- PC: 画面左に常時表示。SP: ハンバーガー等で開くドロワー、または下部タブバー

### メニュー項目（上から）

| 表示名 | 役割 |
|--------|------|
| ホーム | 投稿一覧（トップ）へ |
| 新規投稿 | 新規投稿画面へ |
| プロフィール | 自分のプロフィール・投稿履歴へ |
| 下書き | 下書き一覧へ（任意） |
| 設定 | アカウント・表示設定などへ |
| ログアウト | ログアウト（区切り線の下に配置推奨） |

### 仕様

| 項目 | SP | PC |
|------|-----|-----|
| 幅 | ドロワー時 280px / タブバー時 全幅 | 240px（固定） |
| 背景 | `--sidebar`（= `--card`） | `--sidebar` |
| ボーダー | 右線 0.5px `--border` | 右線 0.5px `--border` |
| 項目の高さ | 48px | 48px |
| 項目の左右パディング | 20px | 24px |
| フォント | 14px, font-weight 500 | 15px, font-weight 500 |

### 配置
- 縦並びリスト。アイコン + ラベル（任意でアイコンのみ SP）
- アクティブ: 背景 `--sidebar-accent`, 文字 `--sidebar-accent-foreground` または `--foreground`
- 非アクティブ: 色 `--muted-foreground`, hover `--foreground`
- ログアウト: 上に区切り線、色 `--muted-foreground` または `--destructive`

### 使用画面
- 全画面。PC では常に左側、SP ではドロワー/タブで表示

---

## 3. パンくずリスト (Breadcrumb)

### 役割
- 現在位置の階層表示（例: Home > Posts > Edit）
- クリックで上位階層へ遷移

### 仕様

| 項目 | SP | PC |
|------|-----|-----|
| 高さ | 40px | 44px |
| 背景 | なし（コンテンツと同じ） |
| 左右パディング | 24px | 48px |
| フォント | 12px(SP) / 13px(PC), 色 `--muted-foreground` |

### 配置
- セパレータ: `/` または `>` 色 `--muted-foreground`
- リンク: 色 `--muted-foreground`, hover `--foreground`
- 現在ページ: 色 `--foreground`, リンクなし

### 使用画面
- 2階層以上ある画面（例: 投稿詳細、編集画面）

---

## 4. フッター (Footer)

### 役割
- コピーライト、利用規約・プライバシーリンクなど
- コンテンツ最下部に配置

### 仕様

| 項目 | SP | PC |
|------|-----|-----|
| 高さ | 最小 64px | 最小 80px |
| 背景 | `--card` または `--background` |
| ボーダー | 上線 0.5px `--border` |
| 左右パディング | 24px | 48px |
| 縦パディング | 24px | 32px |

### 配置
- 中央 or 左揃え
- テキスト: 12px(SP) / 13px(PC), 色 `--muted-foreground`
- リンク: 色 `--muted-foreground`, hover `--foreground`
- リンク間: スペース or `|` で区切り

### 使用画面
- 全画面（メインコンテンツの下）

## レイアウト構成

### PC（横方向）

```
[ サイドメニュー ] | [ ヘッダー（右側エリア上） ]
                   | [ パンくず ]  ※必要な画面のみ
                   | [ メインコンテンツ ]
                   | [ フッター ]
```

### SP（縦方向）

```
[ ヘッダー ]
[ パンくず ]      ※必要な画面のみ
[ メインコンテンツ ]
[ フッター ]

※ サイドメニューはドロワー or 下部タブで表示
```

## カラー・トークン（shadcn / 本プロジェクト dark テーマ）

`frontend/src/app/globals.css` の `.dark` と同期。実装では CSS 変数を参照すること。

| 用途 | CSS変数 | 値（参考・hex） |
|------|---------|-----------------|
| 背景 | `--background` | `#1a1b24` |
| 本文 | `--foreground` | `#fafafa` |
| カード・サイドバー | `--card` / `--sidebar` | `#2d2e3a` |
| カード文字 | `--card-foreground` | `#fafafa` |
| ミュート背景 | `--muted` | `#3f3f46` |
| サブテキスト | `--muted-foreground` | `#a1a1aa` |
| ボーダー | `--border` | `rgba(255,255,255,0.1)` |
| プライマリ（ボタン等） | `--primary` | `#e4e4e7` |
| プライマリ文字 | `--primary-foreground` | `#2d2e3a` |
| 危険（削除等） | `--destructive` | `#dc2626` |

## 角丸（shadcn --radius）

| 用途 | トークン | 値 |
|------|----------|-----|
| 小（input等） | `--radius-sm` | 6px |
| 中（ボタン等） | `--radius-md` | 8px |
| 大（カード等） | `--radius-lg` | 10px |

※ 本プロジェクトの `--radius: 0.625rem`（10px）基準。
