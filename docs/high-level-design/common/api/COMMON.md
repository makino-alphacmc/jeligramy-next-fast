# 共通パーツ API 一覧

`basic-design/COMMON.md` から抜き出した、共通パーツで想定する API。

| API名 | メソッド | パス | 用途 |
|-------|----------|------|------|
| 現在ユーザー取得 | GET | `/api/auth/me` | ヘッダーアバター・メニュー表示 |
| ログアウト | POST | `/api/auth/logout` | サイドメニュー Logout（セッション無効化） |

※ 投稿カードで使う API（投稿一覧・いいね・編集・削除）はトップ等の画面設計を参照。
