# Jeligramy

Instagram風SNSを題材にした学習用プロジェクト。Next.js（フロント）と FastAPI（バック）を分離し、API設計・認証・Docker を学ぶ。

**構成**  
Next.js + TypeScript + Tailwind + shadcn/ui / FastAPI（Python）/ Docker Compose / REST API

---

## 完成形

- 認証（me・logout）、トップは投稿カード一覧（いいね・コメント・本人のみ編集・削除）、DB で永続化

---

## MVP

**ゴール**  
ログイン済み前提で、UI と API がつながり、投稿・いいね・コメントまで動く状態。

| 完了 | 項目                                                                |
| :--: | ------------------------------------------------------------------- |
| [ ]  | 認証（me・logout）— モック → 本実装（セッション/JWT）               |
| [ ]  | 共通レイアウト実装（ヘッダー・サイドメニュー・フッター・パンくず）  |
| [ ]  | ヘッダー：現在ユーザー名表示（GET `/api/auth/me`）                  |
| [ ]  | Logout機能実装：サイドメニューに Logout（POST `/api/auth/logout`）  |
| [ ]  | DB 設計・実装（users / posts / likes / comments）                   |
| [ ]  | 投稿一覧実装（GET `/api/posts`）                                    |
| [ ]  | 投稿の取得・更新・削除機能実装（GET/PATCH/DELETE `/api/posts/:id`） |
| [ ]  | いいね機能実装（POST/DELETE `/api/posts/:id/likes`）                |
| [ ]  | コメント投稿実装（一覧・投稿）                                      |

---

## MVP以降

| 優先度の目安 | 内容                                                       |
